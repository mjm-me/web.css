Scroll animations

Las nuevas propiedades de css permiten una animación basada en el scroll sin necesidad de utilizar JScomo la Intersection Observer API que se utilizaba hasta ahora.

Para ello se utilizan las propiedades view-timeline-name y view-timeline-axis, que definen respectivamente el nombre del timeline y el eje al que se aplica (normalmente block)

En cuanto a la propia animación, queda definida por las siguientes propiedades:

  • animation-name
  • animation-timeline
  • animation-range < indicando entry y cover/li>
  • animation-fill-mode con elñ valor de both

En este ejercicio se ha utilizado la propiedad scroll-snap-type para que el scroll se haga de forma más suave y se pueda ver la animación de los elementos.
